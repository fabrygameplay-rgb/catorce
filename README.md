# 🧫 Bacteria Lab Simulation

Interactive visual experiment that simulates bacterial behavior under different environmental conditions.

Built with HTML, CSS and Vanilla JavaScript.  
Includes dynamic UI themes and real-time visual state changes.

---

## 🧪 Project Overview

This project represents a fictional laboratory environment where bacteria react visually to changing variables.

Each bacterium changes state depending on simulated conditions:

- 🟢 Healthy
- 🟡 Stressed
- 🔴 Critical

The interface includes dynamic screens, variable indicators, and theme switching.

---

## 🚀 Features

- Animated bacteria field
- Dynamic state changes (healthy / stressed / critical)
- Environmental variable simulation
- Multi-theme system
- Fully responsive design (desktop, tablet, mobile)
- Clean layout architecture using Flexbox
- Interactive control buttons
- Visual feedback messages

---

## 🖥️ Technologies Used

- HTML5
- CSS3 (Flexbox, Media Queries, Gradients)
- Vanilla JavaScript
- DOM Manipulation
- Class-based state control

No external libraries or frameworks were used.

---

## 📱 Responsive Design

Breakpoints implemented:

- Desktop (1025px and above)
- Tablet (601px – 1024px)
- Mobile (600px and below)

The layout adapts between:

- Side-by-side structure (desktop)
- Vertical column layout (tablet & mobile)

---

## 🧬 Bacteria System

Each bacterium is rendered as a positioned element inside a simulated Petri dish field.

States are represented visually using:

- Color
- Glow effect
- Transition animations

Transitions are handled smoothly using CSS and JavaScript class toggling.

- Advanced layout control with Flexbox

- Clean separation of structure and style
